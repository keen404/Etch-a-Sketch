# Etch-a-Sketch

## Overview
Etch-a-Sketch เป็นโปรเจกต์ที่นำเสนอเครื่องมือวาดภาพออนไลน์ที่ให้ผู้ใช้สามารถวาดภาพด้วยการเลือกจุดบนกริด โดยโปรเจกต์นี้พัฒนาขึ้นโดยใช้ HTML, CSS และ JavaScript เพื่อเพิ่มประสบการณ์การวาดภาพอย่างสนุกสนาน

## คุณสมบัติ
- **ใช้งานง่าย**: ผู้ใช้สามารถวาดภาพได้เพียงแค่เลื่อนเมาส์ไปยังตำแหน่งที่ต้องการบนกริด
- **ปรับขนาดกริดได้**: ผู้ใช้สามารถเปลี่ยนขนาดของกริดเพื่อเพิ่มความท้าทายในการวาด
- **ล้างภาพ**: มีฟังก์ชันในการล้างภาพเพื่อเริ่มต้นใหม่ได้ตลอดเวลา

## วิธีการใช้งาน
1. เปิดหน้าเว็บของโปรเจกต์
2. เลือกขนาดกริดที่ต้องการ
3. เริ่มวาดภาพโดยเลื่อนเมาส์
4. ใช้ปุ่มล้างเพื่อเริ่มต้นใหม่

## เทคโนโลยีที่ใช้
- **HTML**: ใช้สร้างโครงสร้างพื้นฐานของเว็บ
- **CSS**: ใช้สำหรับการออกแบบและปรับแต่งให้ดูสวยงาม
- **JavaScript**: ใช้สำหรับทำงานกับฟังก์ชันการวาดและจัดการการโต้ตอบของผู้ใช้

## สรุป
Etch-a-Sketch เป็นโปรเจกต์ที่สนุกและสามารถใช้ในการพัฒนาทักษะการเขียนโปรแกรมในระดับเบื้องต้น เป็นเครื่องมือที่ดีในการฝึกฝนการคิดเชิงตรรกะและการสร้างสรรค์จากการวาดภาพ
